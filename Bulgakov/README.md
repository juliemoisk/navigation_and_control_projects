# Задача Булгакова о накоплении возмущений по одной координате

- Исследовала задачу оптимального воздействия возмущений на динамическую систему с целью максимизации отклонения по заданной координате.
- Рассмотрела линейную систему в отклонениях, где возмущение ограничено по модулю, и требуется найти наихудший сценарий его воздействия.
- Сформулировала и решила экстремальную задачу на функциональном пространстве допустимых возмущений, определяя управление, приводящее к максимальному накоплению ошибки в заданный момент времени.
- Реализовала численный алгоритм поиска оптимального (наихудшего) возмущения и интегрирования системы дифференциальных уравнений.
- Показала, что максимальное отклонение достигается при специальном знакопеременном воздействии (аналог «bang-bang» стратегии), согласованном с динамикой системы.
- Провела численные эксперименты, визуализировала траектории и исследовала зависимость максимального отклонения от параметров системы и ограничения на возмущение.
