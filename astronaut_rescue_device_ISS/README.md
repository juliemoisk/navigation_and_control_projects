# Тестирование качества сближения устройства спасения космонавта с МКС

- Исследовала задачу полуавтоматического управления движением в условиях ограниченных ресурсов.
- Изучила алгоритмы стабилизации и сближения устройства с орбитальной станцией.
- Проанализировала взаимодействие автоматического и ручного управления (человек в контуре управления).
- Применяла методы тестирования качества управления при наличии возмущений (максиминный подход).
- Обработала экспериментальные данные движения и визуального поиска цели оператором.
- Освоила методы анализа эргатических систем и управления в космических приложениях.
