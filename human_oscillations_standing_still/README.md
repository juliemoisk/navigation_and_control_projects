# Исследование колебаний человека при спокойном стоянии

- Исследовала систему управления движением человека на примере стабилизации вертикальной позы.
- Использовала модель перевёрнутого маятника для описания биомеханики движения.
- Проанализировала роль сенсорных систем (вестибулярной, зрительной, проприоцептивной) в управлении.
- Выполнила спектральный анализ экспериментальных данных стабилометрии.
- Исследовала влияние внутренних и внешних возмущений на устойчивость системы.
