# Управление возмущаемой нелинейной системой

- Построение управления с единственным переключением для нелинейной управляемой системы второго порядка.
- Разработала алгоритм построения программного управления при отсутствии возмущения и рассчитала фазовую траекторию перехода системы из заданного начального состояния в нулевое.
- Аналитически записала линию переключения управления и реализовала численное интегрирование по этапам с остановкой в момент попадания на линию переключения.
- Провела численное моделирование системы при малом внешнем возмущении и оценила величину промаха по координате y.
