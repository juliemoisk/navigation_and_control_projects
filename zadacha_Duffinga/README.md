# Задача Дуффинга

- Исследовала нелинейную динамическую систему с кубической восстанавливающей силой (осциллятор Дуффинга).
- Выполнила фазовый анализ системы: построение фазовых портретов и исследование устойчивости равновесий.
- Проанализировала влияние параметров нелинейности на характер движения системы.
- Исследовала энергетические характеристики системы и условия существования периодических движений.
- Провела численное моделирование нелинейных колебаний при различных режимах (автономных и вынужденных).
