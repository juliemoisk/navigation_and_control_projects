# Геодезические съёмки с использованием спутниковой навигационной системы

- Выполнила геодезические измерения с использованием GPS/ГЛОНАСС-приемников.
- Обработала экспериментальные данные в среде Matlab.
- Оценила точность позиционирования в стандартном и дифференциальном режимах.
- Применила методы математической статистики для обработки измерений.
- Реализовала алгоритмы оценивания параметров (в том числе метод максимального правдоподобия).
- Освоила принципы работы спутниковых навигационных систем и обработки навигационных данных.
