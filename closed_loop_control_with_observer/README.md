# Синтез стабилизирующего управления для электромагнитного подвеса

- Построение стабилизирующего управления для системы электромагнитного подвеса по измерениям тока в катушке.
- Построила линейную математическую модель движения магнитного диска в окрестности положения равновесия, учитывающую связь координаты, скорости и тока в катушке.
- Синтезировала стабилизирующее управление по неполному измерению, выбрав коэффициенты обратной связи, обеспечивающие асимптотическую устойчивость замкнутой системы.
- Построила замкнутую систему управления и проверила её устойчивость на основе анализа собственных значений.
- Реализовала численное моделирование системы и исследовала поведение системы при наличии возмущений (ошибки измерений и управления).
