# Определение траектории по показаниям инерциальных датчиков

- Реализовала алгоритм решения навигационной задачи для инерциальной навигационной системы (ИНС) по данным акселерометров и гироскопов.
- Построила математическую модель движения объекта с учётом вращения Земли, гравитационного поля и перехода между системами координат (географической и приборной).
- Реализовала алгоритмы:
   – начальной выставки (определение ориентации по показаниям датчиков),
   – счисления ориентации (интегрирование кинематических уравнений),
   – восстановления скорости и координат (интегрирование уравнений движения).
- Разработала численный алгоритм интегрирования матриц ориентации с сохранением ортогональности (формула Родрига) и обработкой дискретных измерений.
- Провела моделирование полёта, обработку данных инерциальных датчиков и сравнение восстановленной траектории с истинной.
